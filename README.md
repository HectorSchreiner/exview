# Logrythm Console Viewer
This is an attempt to create a better and smarter viwer for the logrythm platform. This is not meant to replace the webview panel, but rather create an "admin" panel, to assist in log analysis, and further automate help help where its possible.

# Features
- [ ] Automated drilldowns
- [ ] Automated smart searches, based on the log type
- [ ] Scripting capabilities
- [ ] Possibility for custom automated search queries
- [ ] Implement logging into the comment system in logrythms webconsole ex. `Analyst: "Bob" has initated a search for the ip 192.168.0.1`
- [ ] Automatic Case creation with the given company entity.
- [ ] Integration with compliance automation system
- [ ] Smart alarm handling with keyboardbindings, and no slow ANIMATIONS!
- [ ] Possibly LLM Integration? 
- [ ] Temporary analyst filter creation, to remediate spam.
